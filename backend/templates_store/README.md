# 专利模板文件夹

此文件夹用于存储专利生成的 DOCX 模板文件。

## 模板文件要求

- 文件格式：`.docx` (Microsoft Word 2007 或更高版本)
- 文件编码：UTF-8 (推荐)
- 文件大小：建议不超过 10MB

## 模板内容结构

模板应包含以下章节：

1. **标题** - 专利名称
2. **技术领域** - 所属技术领域描述
3. **背景技术** - 现有技术分析
4. **发明内容** - 核心创新点描述
5. **附图说明** - 图表和说明
6. **具体实施方式** - 详细实施方案
7. **权利要求书** - 专利保护范围
8. **摘要** - 技术方案概述

## 占位符支持

模板支持以下占位符格式：

- `{{标题}}` 或 `{标题}` 或 `<标题>` 或 `[标题]`
- `{{技术领域}}` 或 `{技术领域}` 等

系统会自动将生成的专利内容注入到对应的占位符位置。

## 添加模板

1. 将 `.docx` 模板文件复制到此目录
2. 在前端界面中点击"刷新"按钮重新加载模板列表
3. 选择要使用的模板进行专利生成

## 默认模板

- 如果没有选择特定模板，系统将只生成 Markdown 格式文件
- 系统会自动将第一个有效的模板设为默认模板

## 注意事项

- 确保模板文件内容完整，包含必要的专利章节
- 建议使用标准的专利文档格式
- 模板文件应包含适当的样式和格式设置
